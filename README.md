A modern, highly interactive personal portfolio built with Next.js and React. It features smooth scrolling, custom cursors, and advanced animations to showcase case studies and educational experience.

## Features

- **Interactive Animations** - Complex animations powered by GSAP and Framer Motion.
- **Smooth Scrolling** - Seamless scroll experience implemented with Lenis.
- **Custom UI Elements** - Includes a custom cursor, animated preloader, and tailored design system.
- **Responsive Design** - Fully responsive and optimized for mobile, tablet, and desktop views.
- **Dark Mode Support** - Built-in theming using `next-themes`.

> [!NOTE]
> The design uses modern typography with Clash Display and Satoshi fonts for a clean, professional aesthetic.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Scrolling**: [Lenis](https://lenis.studiofreight.com/)
- **Analytics**: Vercel Analytics & Speed Insights

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Ensure you have Node.js installed (version 20 or higher is recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd ryan-portfolio
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application. The page will auto-update as you edit the files.

### Building for Production

To create an optimized production build:

```bash
npm run build
npm start
```

> [!TIP]
> Use `npm run lint` to catch potential issues early using ESLint.
